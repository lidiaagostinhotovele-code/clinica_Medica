# Clínica Biolife Center — Saúde Premium

**Site institucional da clínica médica Biolife Center**

Breve descrição: A Clínica Biolife Center é uma instituição de saúde premium sediada em Maputo, Moçambique. O website apresenta informações sobre especialidades médicas, equipe, serviços, agendamentos e contactos. O objetivo é oferecer uma plataforma simples e acessível para que pacientes encontrem especialidades, marquem consultas e conheçam mais sobre a clínicas.

---

## 1. Descrição mais detalhada

**Objetivo do site**

* Fornecer informações claras sobre as especialidades médicas oferecidas pela clínica.
* Permitir que visitantes encontrem serviços, contactos, horários, localização e acessem a área de agendamento.
* Reforçar a presença institucional da Biolife Center como clínica premium em Moçambique.

**Público-alvo**

* Pacientes que procuram cuidados médicos especializados.
* Famílias, profissionais e público geral que procuram serviços de saúde confiáveis.
* Pessoas que desejam agendar consultas de forma rápida e prática.

**Conteúdos e funcionalidades oferecidas**

* Página inicial com apresentação da clínica e chamada para ação (Agendar Consulta).
* Listagem das principais especialidades médicas (Cardiologia, Pediatria, Dermatologia, Ortopedia, etc.).
* Seção “Sobre Nós” com informações institucionais.
* Página de agendamento de consultas.
* Informação completa de contacto (telefone, e-mail, endereço).
* Horário de funcionamento atualizado.
* Rodapé com links úteis e redes sociais.
* Destaques de serviços: atendimento de emergência 24h e estacionamento amplo.

> Observação: O site é completamente estático — não processa agendamentos reais. Para implementar marcação de consultas funcional, é necessário integrar API ou back-end.

## 3. Como executar o projeto localmente

Passo a passo:

1. Clone o repositório:

```bash
git clone https://github.com/lidiaagostinhotovele-code/clinica_Medica.git
cd clinica_Medica
```

2. Abra a pasta do projeto no explorador de arquivos ou editor (VSCode, Sublime, etc.).
3. Abra o ficheiro `index.html` diretamente no navegador.

Opcional — iniciar servidor local:

```bash
# Python 3
python -m http.server 8000
# Acesse: http://localhost:8000

# Ou Node.js
npx http-server -p 8000
# Acesse: http://localhost:8000
```
